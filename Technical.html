Form element

Forms are used to submit data to a server, from user inputs. You can do this by specifyiing an action attribute on your form element

The HTML <form> element can contain one or more of the following form elements.

  • <input> -
  • <label>
  • <select>
  • <textarea>
  • <button>
  • <fieldset>
  • <legend>
  • <datalist>
  • <output>
  • <option>
  • <optgroup>


<form id="form" action="https://www.freecodecamp.com/email-submit" method="get">

This will send your email to another server.

Anchor-tag

The anchor tag in HTML can be defined as a means to create a hyperlink that can link your current page on which the text is being converted to hypertext via <a> anchor tag to another page. This anchoring from one page to another is made possible by the attribute href, which can be abbreviated (hypertext reference).


Did you know there is actually several other attributes for the A tag.

  • The href attribute. *The href value is the URL of the linked page.
  • The title attribute. *The title attribute value appears as a tool tip.
  • The rel attribute. *It specifies the relationship between the current document and the linked document.
  • Open the email program. *Open your default email program using the mailto value set to the href property.
  • Open the phone program. *Open your default phone program using the tel value set to the href property.
  • Link to a script. *You can link to a script and execute it with a single click.
  • Link to sections. *You can link to different sections of the same page by linking to elements by id
  • The download attribute. *Turn a link into a download link.
  • The ping attribute. *The ping attribute specifies a space seprated list of URLs to ping when the link is clicked.
  • The type attribute. *The type attribute is used to specify the MIME type of the link target.
  • The target and rel attribute. *The target attribute specifies were to open the document. The rel attribute specifies the relationship between the current document and the linked document.
Internal-links

An internal link will be used to jump to different sections within a page. To create an internal link, you assign a links href attribute to a hash symbol # plus the value of the id attribute for the element that you want to internally link to, usually further down the page. You need to add the same id attribute to the element you are linking to.


Here is an example of an internal link.


<a class="nav-link" href="#form">Form field</a>
<section class="main-section" id="form"></section>
